# Les Douze du Quotidien (French Translation)

French-language edition of my preventive healthcare e-book, originally published in English as **The Daily Dozen**.

## About

This repository hosts the French translation of my e-book on preventive healthcare, produced as part of my AI development training. The translation preserves the structure, page numbering, covers, chapter layout, images, and overall visual style of the original English edition.

- **Author:** Peter N. Ileh
- **Language:** French (translated from English)
- **Format:** PDF
- **Length:** 95 pages, 5 images (excluding front and back cover)
- **Genre:** Preventive healthcare / non-fiction
